# Laura Pastó DAWE
Pràctica RA6 CICD intro
Estructura (HTML): La pàgina s'organitza en una capçalera, una secció amb un formulari per afegir noves tasques i dues llistes dinàmiques diferenciades per a les tasques pendents i les completades.


Disseny (CSS): L'estil s'organitza mitjançant l'ús de variables en el :root per mantenir la coherència visual i utilitza Flexbox per aconseguir un disseny net i totalment adaptatiu (responsive) a dispositius mòbils.


Funcionalitat (JavaScript): El codi s'encarrega de la persistència de dades mitjançant localStorage, gestiona la creació de noves tasques i permet canviar-ne l'estat en fer clic, actualitzant el DOM automàticament.
